Cámara CF7670C-V2

Date dom 13 enero 2013 By Jesús Torres Category Hardware. Tags cf7670c i2c ov7670 al422

Los módulos de cámara CF7670C y CF7670C-V2 combinan el sensor CMOS OmniVision OV7670 con una memoria FIFO Averlogic AL422 de 3Mb.

La idea es que las imágenes capturadas por el sensor se almacenen en la memoria FIFO, dando tiempo a la CPU externa a recuperarlas al ritmo que le sea posible.

Las principales características del sensor OmniVision OV7670 son:

  • Resolución de 640x480 pixels (VGA).
  • Hasta 30 fps para resolución VGA.
  • Formatos de salida (8 bits):
    • YUV/YCbCr 4:2:2
    • RGB 565/555/444
    • GRB 4:2:2
    • Raw RGB Data
  • Modo de escaneo progresivo.

LCD 1.8" SPI (ST7735)

Date dom 13 enero 2013 By Jesús Torres Category Hardware. Tags lcd spi st7735

El módulo en cuestión tiene las siguientes características:

  • Resolución de 128x160 pixels.
  • 262K colores.
  • Retro-iluminación LED.
  • Interfaz SPI a través del chip controlador ST7735R.
  • Compatible con lógica TTL de 3.3 y 5v.

El corazón del módulo es el chip controlador ST7735. Este se conecta por un lado a la LCD propiamente dicha y por el otro ofrece una interfaz SPI a la que se puede conectar una CPU externa. Los datos a mostrar se pueden almacenar en una RAM interna con capacidad para 132x162x18 bits.

Raspberry Pi

Date dom 13 enero 2013 By Jesús Torres Category Hardware. Tags rpi raspberry pi arm gpio videocore broadcom

Una Raspberry Pi es una placa computadora (SBC) de bajo coste que incluye:

  • Un SOC Broadcom BCM2835, que contiene una CPU ARM1176JZF-S a 700 MHz y una GPU VideoCore IV.
  • 512 MB de memoria RAM (en la versión B de Raspberry Pi, 256 MB en la versión A) instalada sobre la CPU.
  • Una ranura de tarjetas SD/MMC/SDIO para el almacenamiento no volátil.

  • 2 puertos USB 2.0.
